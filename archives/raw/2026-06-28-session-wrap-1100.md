## Decisions

- **Session expiry = midnight boundary, not TTL.** Keeps conversational context during a workday, fresh context each morning. (decisions/log.md candidate)
- **Use system clock for day name, not model inference.** Models can't reliably compute day-of-week from a YYYY-MM-DD string.
- **CLAUDE.md staged ingestion rule split:** remote wiki writes stage to `raw/aios/`; working inside claude-os writes directly to curated pages. (decisions/log.md candidate)
- **WIKI-CLAUDE.md "inbox" → "source archive."** Aligned with template and all project wikis. Prevents future raw/ file deletion.
- **Milestone v2.0 closed with tech debt accepted.** Missing VERIFICATION.md files, stale traceability table, deferred UI features (prospect cards, click-to-open). None blocking.
