## Ranked patterns (cross-project weight applied)

### 1. Session-wrap is manual everywhere — ~23 occurrences, ALL repos · fix: hook
"Save all of this to the wiki," "log everything to the wiki," "ingest the latest raw drop," "write a /handoff file," "everything is logged?" — typed over and over in brandos (9), claude-os (7), and client repos (7). When it lapses, it compounds: ToneQuest needed a month of inbox backfill on 6/12. The infrastructure is half-built (`scripts/state-hook`, the unified-wiki design from 6/05, two open todos) and a known blocker is identified: **SessionEnd hook doesn't fire from the IDE extension** (debugged 6/04, tonequest).
**Fix:** session-wrap automation — Stop/SessionEnd hook that stages a session digest to the project wiki `raw/aios/` + writes STATE.md, with a fired/not-fired breadcrumb. Bonus: its distilled output becomes /retro's clean Phase-2 input. **← #1 candidate for next /level-up.**

### 2. MCP availability churn — ~16 occurrences, brandos + 3 client repos · fix: rule + doctor
"Is the DO MCP active?" asked verbatim across 5 fresh sessions in 45 minutes on 6/08 (with restarts between). Same pattern for Dropbox, Higgsfield, Shopify dev MCP.
**Fix:** rule — new MCPs go global in `~/.claude.json` with `${ENV_VAR}` refs, never inline tokens, verified with a test call before moving on. Plus an MCP-status line in each project CLAUDE.md.

### 3. Design output misses brand/spec, full redo rounds — 9+ msgs, brandos · fix: spec + gate
Wrong branding (didn't crawl terraplexag.com), contrast/readability misses, "doesn't feel like BrandOS," marquees recurring.
**Fix:** per-tenant DESIGN-SPEC.md (brand tokens, source-of-truth URL, banned patterns, WCAG gate) every design agent must load; automated contrast check before presenting. Belongs in the brandos repo.

### 4. Re-orientation tax at session start — ~16 msgs, brandos + claude-os · fix: hook
"Where did we leave off on MM?", "what GSD phase are we in?", "where did that research go?" Many fresh windows per day, each paying the tax.
**Fix:** SessionStart banner per repo — GSD position, open spikes, last wiki log entry, where-things-live map. (claude-os side largely addressed by the 6/12 router rebuild; project repos still need it.)

### 5. Token-budget babysitting — 6 msgs, claude-os · fix: rule
"I'm at 57%... let's test the waters," "that did nothing to my usage, go ahead with the next 2." Justin is the budget governor for background execution.
**Fix:** budget-aware execution rule — agent reports usage % after each wave and auto-continues below a set threshold; calibration already exists in `references/claude-code-token-economics.md`.

### 6. Video ingestion negotiated ad-hoc every time — 6 msgs, claude-os · fix: skill

Partially absorbed today: the transcript-fetch logic from the Zack Proser ingest (innertube API + timedtext XML parse) is proven and sitting in this session — worth hardening into `/ingest-video` (URL → transcript → digest → routed per /dispatch rules).

### 7. Dev-env churn: orphan ports + supabase by hand — ~16 msgs, brandos · fix: scripts
