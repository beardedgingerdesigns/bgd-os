<objective>
Build the Receipt feed foundation: the `lib/raw-drops.ts` helper used by every subsequent write surface, and the receipt persistence + UI dock that every later plan will emit into.

Purpose: Without this scaffold, no other plan can prove its writes are happening. The receipt feed IS the trust mechanism for the phase (HUB-06). Per ADR 0003 Decision §"Outputs propagated": "Every write emits a Receipt feed entry (timestamp, kind, project, absolute file path, excerpt)."

Output: A fully wired `lib/raw-drops.ts` module, append-only receipts persistence, GET endpoint, global dock UI, layout slot, gitignore update, and unit tests. No producers of receipts yet — those are wired by Wave 2+ plans.
</objective>
